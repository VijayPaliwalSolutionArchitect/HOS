# Place images here

## Required Images

1. **icon.png** (1024x1024)
   - The main app icon
   - Used for iOS and Android
   - Should be a square PNG with transparent or solid background

2. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon foreground
   - The icon should be centered in a 1024x1024 canvas
   - Keep important elements in the center 768x768 safe zone

3. **splash.png** (1242x2436 or larger)
   - The splash screen image
   - Will be displayed while the app is loading
   - Use a centered logo on a solid background

4. **favicon.png** (32x32)
   - Web favicon for PWA
   - Small icon for browser tabs

## Creating Icons

You can use tools like:
- Figma or Adobe Illustrator for design
- https://www.appicon.co/ for generating all sizes
- https://icon.kitchen/ for adaptive icons

## Image Optimization

Consider optimizing images before adding them:
- Use PNG for icons and logos
- Use WebP for photos (with PNG fallback)
- Compress images using tools like TinyPNG or ImageOptim
